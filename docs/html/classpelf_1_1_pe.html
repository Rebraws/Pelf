◆ getSections()

template<class Container , std::size_t NumOfSections>
constexpr auto pelf::Pe< Container, NumOfSections >::getSections
constexprnoexcept

Returns the section table from the Pe file.

If the parsing is done at compile time it returns an array with all the sections from the Pe file, each section is represented by the hana struct IMAGE_SECTION_HEADER, on the other side, if parsing happens at runtime it returns a std::vector with all the sections.

Returns
Returns either an array or a vector that represents the section table
